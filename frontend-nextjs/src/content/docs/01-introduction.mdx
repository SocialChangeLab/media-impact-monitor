# What is the Media Impact Monitor?

The **Media Impact Monitor** is an innovative tool designed for protest groups and NGOs to assess their influence on public discourse. By analyzing data from multiple media sources, including newspapers, social media, and parliamentary debates, the tool provides a comprehensive overview of how activist events shape media narratives and public agendas.

To learn more about the project, visit the [About](/about) page.

## Key Features

-  **Quantitative and Qualitative Analysis**: Measure the impact of your actions through detailed metrics and visualizations.
-  **Multi-Source Integration**: Analyze data from local and national newspapers, social media platforms, and more.
-  **Causal Inference Methods**: Utilize advanced statistical techniques to understand the true impact of your events on public discourse.
