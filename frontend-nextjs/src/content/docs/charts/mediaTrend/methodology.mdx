---
title: Methodology of this chart
date: 2024-06-26
---

1. Data Collection: Media articles mentioning protest events are gathered.
2. Sentiment Analysis: Each article undergoes a sentiment analysis process using natural language processing (NLP) techniques to determine its sentiment score.
3. Averaging Sentiment: For each protest event, the sentiment scores of all related articles are averaged to provide an overall sentiment score.
4. Visualization: The averaged sentiment scores are plotted on the chart for the selected time period, providing a clear view of media sentiment trends.
