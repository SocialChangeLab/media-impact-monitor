## Sources

The data used in this chart provides from the following sources:

- Media Cloud
- Genios
- Nexis
- DeReKo (Deutsches Referenzkorpus)

## Data processing

The data used in this chart is processed as follows:

- The data is filtered to only include protest events that occurred in the selected time period.
- The data is filtered to only include articles that were mentioned in the protest events.

## Data limitations

There are some limitations to the data used in this chart:

- The data is filtered to only include protest events that occurred in the selected time period.
- The data is filtered to only include articles that were mentioned in the protest events.
