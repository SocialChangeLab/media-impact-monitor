---
title: How to read this chart
date: 2024-06-26
---

The chart aggregates sentiment measurements of articles related to each protest event:

- Positive Sentiment: Indicates favorable media coverage.
- Negative Sentiment: Indicates unfavorable media coverage.
- Neutral Sentiment: Indicates impartial media coverage.
- Trend Lines: Represent the general sentiment trend over time for the chosen period.

To interpret the chart:

1. Select the desired time period.
2. Observe the average sentiment value for each protest event within this period.
3. Pay attention to upward or downward trends in sentiment to understand how media perception changes over time.
